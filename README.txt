ArcheoCrop version 1.0

Приложение для удаления фона у фотографий. Создавалось как приложение для археологов
для удаления фона у фотографий археологических находок для их последующей обработки и 
подготовке к публикации\отчету.

Может работать не только с фотографиями артефактов, но в первую очередь именно для артефактов.

Также приложение позволяет установить кастомный фон, значения для которого в формате RGBA
можно задать самостоятельно

У приложения есть два сценария обработки изображений - один стоковый, самый обычный (он менее 
ресурсозатратный, но и менее качественный) и продвинутый (более ресурсозатратный, более качественный)



Принцип работы:
запускаете exe файл.
выбираете вариант работы (с одним изображением или с архивом)
указываете путь к целевому файлу
указываете фон при необходимости
указываете продвинутый сценарий обработки (при необходимости)
полная пошаговая инструкция по работе + требования к фото в файле user_guide

функционал реализован при помощи библиотеки rembg
